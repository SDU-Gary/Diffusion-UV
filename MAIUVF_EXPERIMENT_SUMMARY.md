## 💡 对 Phase 2 的建议

### 🔥 紧急改进 (优先级 1)

**问题**: 法向梯度浪费容量

**解决方案**: 切空间投影
```python
# 在 L_metric 计算中添加投影
J_tangent = J - (J @ n) @ n^T
L_metric = ||J_tangent_pred - J_tangent_gt||^2
```

**预期收益**: 
- 减少 ~20% 容量浪费
- 提升 UV 预测精度
- 更符合流形参数化理论

### 📈 短期优化 (优先级 2)

**问题**: 缺少法向约束

**解决方案**: 添加法向正则化
```python
L_normal = ||J @ n||_2
L_total = L_metric + L_anchor + λ * L_normal
```

**预期收益**:
- 抑制不必要的法向学习
- 提高流形对齐度

### 🔬 中期研究 (优先级 3)

**探索方向**:
- Fourier vs B-Spline 对比实验
- 混合编码方案
- 多尺度特征融合
