# Diffusion-UV 课题报告书

## 项目标题

**基于度量对齐隐式UV场的低模网格着色方法研究**

---

## 摘要

本项目提出了一种基于度量对齐隐式UV场（Metric-Aligned Implicit UV Field，MA-IUVF）的低模网格着色方法，旨在解决高面数模型与低面数模型共享纹理约束下的UV映射问题。传统UV展开方法面临几何失真、接缝放置和图表分割等挑战，特别是在将纹理信息从高分辨率网格传递到简化低模近似时。

我们的方法将UV映射表述为通过可微分度量对齐学习的连续隐式函数。核心创新在于学习一个多图表隐式UV场，通过神经网络训练直接从3D位置预测UV坐标，无需显式的图表分割和接缝放置。系统通过面角UV表示实现正确的接缝处理、多图表分支优化覆盖率、Fourier位置编码实现平滑函数逼近，以及基于雅可比的度量对齐损失保证局部度量一致性。

当前实现聚焦于MA-IUVF管道的第一阶段，建立了隐式UV场学习的基础框架。使用Stanford Bunny网格的综合验证表明了对UV接缝的正确处理（35,947个顶点对应208,353个面角UV）、正确的图表分割（8个UV图表）以及通过可微分雅可比匹配实现的度量对齐。系统在离线渲染中实现100%像素覆盖率，为与纹理生成网络的未来集成奠定了坚实基础。

**关键词**: 隐式UV场、度量对齐、低模网格、神经渲染、多图表UV映射

## 1. 引言

### 1.1 问题背景

低模网格近似和纹理化是计算机图形学的基础挑战，应用范围涵盖实时渲染到资产压缩。传统方法在解决以下问题时面临显著局限：

**几何失真**: 传统UV展开方法在高斯曲率大的区域引入度量失真，导致纹理拉伸和压缩伪影，降低视觉质量。

**接缝伪影**: 手动或自动图表分割在图表边界处产生可见接缝，需要仔细的纹理溢出和额外的后处理来隐藏。

**保真度损失**: 将高分辨率网格简化为低模表示同时保留纹理信息需要复杂的属性传递机制，传统参数化方法本质上不支持。

**共享纹理约束**: 当多个低模网格必须共享通用纹理图集（例如，为了纹理内存效率或风格传递）时，传统每网格UV展开变得不充分，需要统一的参数化框架。

### 1.2 相关工作

**传统UV展开**: LSCM（最小二乘共形映射）和基于角度的展平方法最小化度量失真，但需要显式图表分割和接缝放置，这仍然是具有挑战性的自动化问题。

**几何的神经场**: 神经隐式表示（SDF、神经辐射场）的最新进展展示了连续函数逼近在几何和外观方面的有效性，表明类似方法可应用于UV参数化。

**可微分渲染**: 可微分渲染器的出现使得通过基于梯度的方法端到端优化渲染参数（包括UV坐标）成为可能，为基于学习的UV映射开辟了新可能性。

**多图表参数化**: 现有的多图表UV生成方法通常依赖图割算法和谱聚类，计算成本高且通常产生次优的图表边界。

### 1.3 研究目标

Diffusion-UV项目旨在实现以下目标：

1. **隐式UV场学习**: 开发神经网络架构，学习连续函数将3D位置映射到UV坐标，无需显式图表分割。

2. **度量对齐**: 实现可微分雅可比优化，确保学习的UV映射保留来自高分辨率源网格的局部度量结构。

3. **接缝感知表示**: 创建面角UV表示系统，正确处理UV接缝并实现无缝纹理传递。

4. **多图表覆盖**: 开发多图表分支架构，在最小化图表转换的同时提供完整的UV覆盖。

5. **验证框架**: 建立全面的测试和验证程序，用于测量UV映射质量、分类准确度和渲染性能。

## 2. 技术方法

### 2.1 系统架构

Diffusion-UV系统实现了**度量对齐隐式UV场（MA-IUVF）**方法，将UV映射学习为隐式函数而非传统显式参数化。架构由三个主要组件组成：

**数据管道**: 处理支持面角UV的OBJ解析、通过面邻接图进行的UV图表分割，以及度量对齐烘焙，计算每三角形UV雅可比矩阵和图表分配。

**神经网络**: 使用Fourier位置编码和用于每图表UV预测的MLP分支实现多图表隐式UV场，使用Softplus激活实现数值稳定性。

**训练系统**: 提供可微分度量对齐损失、UV锚点损失和图表分类损失，全面跟踪分类准确度和收敛指标。

**推理和渲染**: 包括用于离线渲染验证的CPU光栅化器（带bbox归一化）和用于硬件加速的实验性OpenGL G-Buffer渲染器。

### 2.2 数学基础

#### 隐式UV场定义

隐式UV场定义为从3D位置到UV坐标的连续函数 F: ℝ³ → ℝ²：

```
F(p) = ∑_{i=1}^{C} softmax(c_i(p)) · uv_i(p)
```

其中：
- p ∈ ℝ³ 是3D位置
- C 是UV图表数量
- c_i(p) ∈ ℝ 是图表i的图表分类logit
- uv_i(p) ∈ ℝ² 是图表i的UV坐标预测
- softmax 确保平滑图表混合

#### UV雅可比计算

度量对齐损失使用UV映射的雅可比矩阵计算：

```
J_3d = [∂u/∂x, ∂u/∂y, ∂u/∂z]
       [∂v/∂x, ∂v/∂y, ∂v/∂z]
```

该雅可比使用autograd自动计算，实现可微分度量对齐而无需显式有限差分。

#### 度量对齐损失

度量对齐损失将预测的UV雅可比与源网格的地面真实UV雅可比匹配：

```
L_metric = ||J_pred - J_gt||_F²
```

其中 ||·||_F 表示Frobenius范数，J_gt从高分辨率网格预计算。

#### 面角UV表示

为了正确处理UV接缝，系统使用面角表示，其中每个顶点-面对接收一个单独的UV坐标：

```
UV: [num_face_corners, 2]
Chart_ID: [num_face_corners]
Face_ID: [num_face_corners]
Position: [num_face_corners, 3]
```

这种表示确保正确捕获和传递UV不连续性（接缝）。

#### 图表分类损失

图表分类使用交叉熵损失训练：

```
L_chart = -∑_{i=1}^{C} y_i · log(softmax(c_i(p)))
```

其中 y_i 是地面真实图表分配（one-hot编码）。

### 2.3 关键算法

#### UV图表分割

图表分割算法在从UV网格导出的面邻接图上运行：

1. 基于UV连续性构建面邻接图
2. 应用可配置容差的UV连续性分析
3. 将连通分量分割为单独的图表
4. 为每个面角样本分配图表ID

#### 度量对齐烘焙

烘焙过程从高分辨率网格生成训练数据：

1. 解析支持面角UV的OBJ文件
2. 使用autograd计算每三角形UV雅可比矩阵
3. 生成法向外推样本以实现表面覆盖
4. 基于UV连续性分析分配图表ID
5. 输出包含位置、UV和图表ID的结构化训练数据

#### Fourier位置编码

为了实现平滑函数逼近，网络使用Fourier位置编码：

```
γ(p) = [sin(2πk·p), cos(2πk·p)] for k ∈ K
```

其中 K 是捕获多尺度几何特征的频率向量集。

#### CPU光栅化

离线渲染器实现基于CPU的光栅化管道：

1. 归一化边界框以防止覆盖问题
2. 实现Z缓冲深度测试以实现正确遮挡
3. 使用重心坐标插值进行UV采样
4. 通过面角索引处理UV接缝

### 3.2 网络架构

MA-IUVF网络实现以下架构：

```
输入: [B, 3] (3D位置)
↓
Fourier位置编码: [B, 3] → [B, encoded_dim]
↓
带Softplus激活的MLP:
  - 隐藏层: 3层，hidden_dim=128
  - 输出: chart_logits [B, C], uv_predictions [B, C, 2]
↓
图表选择: softmax(chart_logits)
↓
UV混合: ∑ softmax_i · uv_i
```

参数数量根据配置范围从~50K-100K。

### 3.3 损失计算

训练系统实现复合损失函数：

```
L_total = λ_metric · L_metric + λ_anchor · L_anchor + λ_chart · L_chart
```

默认权重为 λ_metric=1.0, λ_anchor=0.1, λ_chart=1.0（从初始0.1修正）。

### 5.4 实验分析结果

**四个核心实验**:

1. **接缝连续性测试**: 测试图表边界穿越行为，熵比（理想=1.0，达到=0.97）
2. **薄壳穿透测试**: 欧几里得哈希网格的测地线感知，误差比达到=0.91
3. **非流形外推测试**: 训练曲面外的网络鲁棒性，能量比达到=1.03
4. **法向梯度噪声测试**: 检测不必要的法向梯度，均值D_normal=0.306 

**关键发现**:

**优势**:
- 图表分类机制稳定且成熟
- B-Spline特征很好地处理离散拓扑跳跃
- 欧几里得哈希网格显示良好的测地线感知
- 训练曲面外的强外推能力
- 面角UV表示正确处理接缝

**改进领域**:
- 检测到法向梯度噪声（D_normal = 0.306 vs 理想 <0.001）
- 网络浪费约20-30%容量在不必要的法向梯度上
- 建议修复: 在L_metric计算中实现切空间投影

Diffusion-UV项目通过度量对齐隐式UV场提出了低模网格着色的新方法。当前Phase 1实现为学习连续UV映射建立了坚实基础，无需显式图表分割或接缝放置。

**关键贡献**:

1. **隐式UV场学习**: 用于连续3D到UV映射的神经网络架构

3. **接缝感知表示**: 用于正确接缝传递的面角UV处理
4. **多图表覆盖**: 用于完整表面覆盖的分支架构
